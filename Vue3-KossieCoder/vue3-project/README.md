# vue3-project

## Vue3 install
```
npm install -g @vue/cli
```

## Vue3 version
```
vue vue -V
``` 

## Vue3 add project
```
vue create vue3-project
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
